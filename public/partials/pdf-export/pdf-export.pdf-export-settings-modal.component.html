Settings for PDF export

HTML template for the print header. The default is to show the date on the right.
HTML template for the print footer. The default is to show the page number and the total number of pages on the right.
Header and footer templates should be valid HTML markup with following classes used to inject printing values into them:
  • date formatted print date
  • title document title
  • url document location
  • pageNumber current page number
  • totalPages total pages in the document
